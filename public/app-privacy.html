微企聊隐私保护指引

导言

微企聊（以下简称“本应用”）是由企安信（杭州）科技有限公司（以下简称“我公司”）开发的一款私有化部署的即时通讯软件。为说明本应用如何收集、使用和存储您的个人信息，以及您享有何种权利，我们将通过本指引向你阐述相关事宜，其中要点如下：

  1. 我们将明确本应用中涉及的信息收集方（包括我公司及使用微企聊的企业运营者），以便您了解各方收集信息的范围及原因。

  2. 我们将逐一说明我们收集的您的个人信息类型及其对应的用途，以便您了解我们针对某一特定功能所收集的具体个人信息的类别、使用理由及收集方式。

  3. 当您使用某些功能时，我们会在获得您的同意后，收集您的设备信息。除非按照相关法律法规要求必须收集，拒绝提供这些信息仅会使您无法使用相关特定功能，但不影响您正常使用本应用的其他功能。

  4. 我们不会主动共享或转让您的个人信息至除我公司、使用微企聊的企业运营者外的第四方。如存在其他共享或转让您的个人信息，或您需要我们将您的个人信息共享或转让至第四方情形时，我们会直接征得或确认第四方征得您对上述行为的明示同意，但法律法规另有规定或因保护用户权益或保护本应用生态安全需要除外。此外，我们会对对外提供信息的行为进行风险评估。

  5. 为了在本指引下收集您的信息，或者向您提供服务、优化我们的服务以及保障您的账号安全，我们将需要向您索取相关的手机设备权限；其中的敏感权限例如摄像头、麦克风、相册等均不会默认开启，只有在您明确同意后我们才会在您同意的范围内调用或使用。对您已经向我们授权的该等权限，可以在手机的系统设置中查询我们已调用或使用的系统权限类型、使用目的，并进行统一管理。

需要特别说明的是，我们获得具体某一项敏感权限并不代表我们必然会收集您的相关信息；即使我们已经获得敏感权限，也仅在必要范围内，并在您使用相关的服务或功能期间根据本指引来收集您的相关信息。

若您希望快速了解收集的信息及第三方信息共享情况，请查看《微企聊个人信息收集清单》和《微企聊第三方信息共享清单》

一、信息的收集方说明

微企聊是一款企业级的私有化部署即时通讯软件，企业用户在向我公司购买该软件后，将自行负责其运营。在运营过程中，企业会收集使用本软件的用户信息，包括手机号、昵称、头像等。

作为本软件的开发商和技术维护方，我公司仅提供软件销售及售后技术支持。在维护过程中，为确保软件的正常运行，我们将收集您的设备信息和IP地址。此外，软件内置的投诉功能允许用户主动提交投诉，只有在您填写并提交相关信息后，我公司才能获取这些用户信息。

企业客户在购买我公司的软件时，需遵循国家相关法律法规，提供手机号和邮箱。这些信息将用于客户联系及新产品推广通知（通过短信或常用通讯工具、邮件等方式）。如果企业客户希望取消后续的推广通知，可以随时与我公司的客服团队联系。

二、我们如何收集、使用信息

在您使用本应用的过程中，我们会按照如下方式收集、使用您在使用服务时主动提供或因为使用服务而产生的信息：

2.1 当您注册本应用时，企业的运营方会收集、使用您手机号码并保存在企业的服务器上。收集这些信息是为了帮助您完成本应用注册和使用本应用通信等功能，保护您本应用账号的安全。收集、使用手机号码是为了满足相关法律法规的真实身份信息认证要求。若您不提供这类信息，您可能无法正常使用我们的服务。您还可以根据自身需求选择填写性别信息。

当您主动提交投诉时，我公司作为软件的开发商和技术维护方，我们会收集您的昵称、手机号码信息，收集这些信息是为了判断投诉的真实性，并向您反馈投诉的处理结果。

2.2 当您使用本应用服务时，为保障您正常使用我们的服务，维护我们服务的正常运行，改进及优化我们的服务体验以及保障您的账号安全，我们会收集、使用您的设备信息（设备序列号、设备型号、操作系统、唯一设备标识符如 Android ID、OAID、IDFA、GUID 或 MAC 地址、IMEI、IMSI、SIM 卡序列号）、登录 IP 地址、本应用软件版本号、接入网络的方式、类型和状态、网络质量数据、操作日志、服务日志信息等日志信息。同时，为了能够正常打开文件，我们可能会收集您设备安装的应用信息。

2.3 为向您提供消息离线推送功能，本应用内置主流安卓手机厂商的推送 SDK，各 SDK 可能会收集您的设备信息，例如 Android ID、OAID 等。不同 SDK 收集的信息具体请见附录：《第三方 SDK 设备信息收集明细》

2.4 您在个人信息中填写的性别、昵称、头像等信息将被上传并存储在企业运营方的服务器中，以便您在所有设备登陆时可以同步个人信息。您有权随时修改这些信息。拒绝提供该信息仅会使您无法在个人信息里体现头像、昵称和性别，不影响您正常使用本应用的其他功能。

2.5 当您使用本应用的群聊功能时，群聊名称、群公告、群设置、备注等信息会存储在我们的服务器中，这是实现这一功能所必需的。你的头像、昵称等还会向群成员展示。

2.6 为确保您能够顺利使用本应用的即时通信功能，以下是需要您授权的权限说明：

  • 联网权限：本应用需要联网以进行注册、登录及消息的收发。如果不授权此权限，您将无法正常使用应用。

  • 通知权限：为了在您未使用本应用时及时接收消息提示，我们将请求通知权限。未授权此权限不会影响应用的主要功能，但会影响新消息的提示。

  • 相机权限：本应用需要相机权限以便在会话中拍照、录制视频、设置头像和扫码。未授权此权限不会影响应用的主要功能，但会限制拍照、发图和扫码等功能。

  • 相册与存储权限：为了让您能够从相册选择图片发送、设置头像、保存照片及发送文件，我们将请求相册与存储权限。未授权此权限不会影响应用的主要功能，但会影响从相册发图和设置头像等功能。

  • 麦克风或录音权限：本应用需要麦克风权限以便发送语音消息和录制视频。未授权此权限不会影响应用的主要功能，但会限制相关的语音和视频功能。

  • 应用自启动和关联启动权限：为了确保在关闭或后台运行时能够接收到消息通知，本应用需要自启动权限。未授权此权限只会影响消息通知，不会影响其他功能的使用。

  • 传感器权限：在播放视频等内容时，应用可能会调用设备的重力传感器和加速度传感器，以适应设备的横竖屏状态。

2.7 我公司不会收集您的聊天记录。因为本应用是私有化部署的即时通信软件，您在本应用中和好友的聊天记录会加密传输和存储在由企业运营的服务器中。您在本应用中的聊天记录会存储在您和好友的终端设备，您可以选择自行备份、删除或在不同设备中转移。

2.8 如果您联系我们的客服，我们可能需要您提供必要的个人信息进行身份验证，以保证您的账号安全。为根据您的诉求提供服务，经过您的授权，人工客服人员需要在您授权范围内查询或核验您的相关信息，我们将尽可能采取技术和管理措施保障您的信息安全和保密，并在必要范围内进行使用。我们可能还会保存您的联系方式（您使用的或主动提供的手机号码、本应用的服务器ID、账号、其他通讯工具的账号或其他联系方式）、您与我们的通信记录和内容，以及其他必要信息，以便为您提供和记录客户服务 。

2.9 另外，根据相关法律法规及国家标准，以下情形中，我们可能会处理您的相关个人信息而无需征求您的授权同意：

  • 为个人信息处理者履行法定义务或法定职责所必需；

  • 为订立、履行您作为一方当事人的合同所必需；

  • 为应对突发公共卫生事件，或者紧急情况下为保护自然人的生命健康和财产安全所必需；

  • 依法在合理的范围内处理您自行公开或者其他已经合法公开的个人信息；

  • 法律、行政法规规定的其他情形。

2.10 为了确保服务的安全，帮助我们更好地了解我们应用程序的运行情况，我们会使用腾讯 Bugly SDK 记录相关日志信息，例如，您使用应用程序的频率、崩溃数据、总体使用情况、性能数据以及应用程序的来源。我们不会将我们存储在分析软件中的信息与您在应用程序中提供的任何个人身份信息融合处理。收集的信息具体请见附录：《第三方 SDK 设备信息收集明细》之“（6）Bugly SDK”

2.11 当您在本应用中复制和粘贴信息时，我们将申请使用您的剪切板，我们仅在本地识别剪切板内容不会上传您剪切板的信息至我们的服务器。

2.12 如我们使用你的个人信息，超出了你所授权的目的及具有直接或合理关联的范围，我们将在使用你的个人信息前，再次向你告知并征得你的明示同意。

此外，我们将会严格遵守相关法律法规的规定，并要求第三方保障其提供的信息的合法性。

三、我们如何存储个人信息

3.1 信息存储的地点

我们会按照法律法规规定，将境内收集的用户个人信息存储于中国境内。如果您的个人信息存储地点从中国境内转移到境外的，我们将严格依照法律的规定执行。由于本应用是一款私有化部署的即时通信软件，所有信息将存储在企业自有的服务器上，由我公司负责后续的技术维护和支持。这样可以确保数据的安全性和隐私性，同时提供更高效的服务保障。

3.2 信息存储的期限

一般而言，我们仅在为实现目的所必需的时间及法律、行政法规所规定的保存期限内保留您的个人信息，例如：

手机号码：无论您是通过手机号码注册还是已有企业运营者为您在管理后台创建的账号，若您需要使用本应用服务，企业服务器会一直保存您最后绑定的手机号码，以保证您正常使用该服务，您可以更换绑定的手机号；

头像：您主动设置的用户头像；

昵称：您主动设置的用户昵称；

聊天信息：加密保存且无法在服务器上被查看。

当您注销本应用账号后，企业服务器将依法删除相关信息。在企业服务器停止运行时，我公司将根据法律法规的要求，保留服务器上的信息至规定时长。如果企业服务器发生停止运营的情况，我们将通过系统通知、短信等方式及时告知您。在数据保存期限结束后，我公司将删除您的个人信息或进行匿名化处理。（本指引中的“匿名化处理”，是指通过对个人信息的技术处理，使得个人信息主体无法被识别，且处理后的信息不能被复原的过程。个人信息经匿名化处理后所得的信息不属于个人信息。）

四、我们如何保护个人信息的安全

我们努力为用户的信息安全提供保障，以防止信息的丢失、不当使用、篡改、未经授权访问或披露。

我们将在合理的安全水平内使用各种安全保护措施以保障信息的安全。例如，我们会使用加密技术（例如，SSL）、去标识化、匿名化处理等手段来保护您的个人信息。

我们通过不断提升的技术手段加强安装在您设备端的软件的安全能力，以防止您的个人信息泄露。消息将进行加密保存，并在服务器之间进行加密传输，以确保数据的安全性和隐私性。

若发生个人信息泄露等安全事件，我们会启动应急预案及补救措施，阻止安全事件扩大，并依法履行相关的报告及通知义务。

我们会尽力保护您的个人信息。 我们也请您理解，您连接到企业服务器所使用的系统和通讯网络，可能会因我们无法控制的因素而出现问题，任何安全措施都无法做到无懈可击。

五、我们可能分享的信息

5.1 我们不会向合作伙伴分享可用于识别您个人身份的信息，除非您明确授权。

5.2 为确保部分独立功能的实现，使您能够使用和享受更多的服务及功能，我们会在应用中嵌入第三方的 SDK，我们将审慎评估该等 SDK 的使用目的。了解更多有关 SDK 处理个人信息的目的、方式及范围等信息，请查看文末附录：《第三方 SDK 设备信息收集明细》。

随着我们业务的持续发展，我们有可能进行合并、分立等交易，若涉及个人信息转移的，我们将告知您相关情形，按照法律法规及不低于本指引所要求的标准继续保护或要求新的处理者继续保护您的个人信息。

六、您如何访问及控制自己的个人信息

在您使用本应用期间，为了您可以更加便捷地查阅、复制、更正、删除您的个人信息，同时保障您撤回对个人信息处理的同意及注销账号的权利，我们在产品设计中为您提供了相应的操作设置，您可以参考下面的指引进行操作。此外，我们还在应用给设置了投诉的渠道，您的意见及请求将会得到及时的处理。

6.1 查询、修改个人信息

您可以在本应用内访问“我的”页面，点击顶部的个人信息卡片，查询并修改相关个人信息。

6.2 删除个人信息

您可以在在进入本应用的任一单聊或群聊后，点击聊天右上角的图标，进入聊天详情页点击“清空聊天记录”，删除对应的聊天记录。或打开会话，长按一条需要删除的消息，选择“删除”，删除此条消息。

6.3 系统权限管理

您可以在手机“设置”中找到“应用管理”，找到本应用并打开应用详情，可进行对应的系统权限设置。

6.4 注销账号

您可以在应用内访问“设置”-“账户与安全”-“注销账号”发起账户注销，注销时需使用账号注册手机号的短信验证码进行身份验证，验证通过后即可注销，立即生效。或通过联系企业运营方，由企业运营方在管理后台注销账户。

当您注销账号后，我们将按照《中华人民共和国网络安全法》等法律法规的规定留存您的相关信息；超出必要保存期限后，我们将删除或匿名化处理您的个人信息。

6.5 撤回已同意的授权

您可以通过本应用在您设备初次运行时的权限申请提示窗口，或者修改设备设置中本应用对应的权限设置来撤销对我们的授权，但您的拒绝授权行为可能会导致相关应用功能不可用。

6.6 投诉举报

本应用不涉及任何算法在内的信息系统自动决策机制，所有涉及用户信息的操作都由服务运营方在后台系统手工管理。 如果您在使用本应用的过程中，因账号封禁、注销等企业运营方操作原因无法使用，请与客服团队联系进行反馈或申诉。

如果您在使用应用的过程中，遇到其他用户和群组中的违规行为，您可以点击聊天右上角的图标，进入聊天详情页找到“投诉”，填写投诉信息后提交投诉。

七、变更

请您理解，我们向您提供的功能和服务是不断更新和发展的，如果某一功能或服务未在前述说明中告知收集、使用您的信息的处理规则，我们可能会适时对本指引进行修订。当指引的条款发生重大变更时，我们会在您登录及版本更新时以推送通知、弹窗或其他符合法律要求的适当形式向您告知变更后的指引，以征得您的同意。

八、未成年人个人信息的保护

我们非常重视对未成年人个人信息的保护。根据相关法律法规的规定，若您是 18 周岁以下的未成年人，在使用本应用前，应事先取得您的家长或法定监护人的书面同意和指导。若您是未成年人的监护人，请您先阅读并充分理解本隐私保护指引，当您对您所监护的未成年人的个人信息有相关疑问时，请通过下文第 9 部分中的联系方式与我们联系。

特别地，若你是14周岁以下的儿童，我们还专门为你制定了《儿童隐私保护声明》，儿童及其监护人在为14周岁以下的儿童完成账号注册前，还应仔细阅读我公司专门制定的《儿童隐私保护声明》。只有在取得监护人对《儿童隐私保护声明》的同意后，14周岁以下的儿童方可使用本应用的服务。
