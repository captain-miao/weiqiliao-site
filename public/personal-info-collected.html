微企聊（下称“本应用”）将收集您在使用应用和服务过程中主动提供、授权后通过程序自动化收集的信息。为了帮助您更好地了解和感知我们在各业务功能和场景下，对个人信息收集、使用情况，我们特整理了本清单：

使用场景：用户进行注册、登录、修改密码 个人信息内容：手机号码；密码 使用目的：注册和登录账号 收集方式：用户主动提供

使用场景：用户完善账号信息时 个人信息内容：账号；昵称；头像；性别 使用目的：完善账号信息 收集方式：用户主动提供

使用场景：用户使用账号时 个人信息内容：账号；注册、登录、修改设置的操作记录IP；安全设置 使用目的：保障注册、登录帐号安全 收集方式：自动化收集；用户主动提供

使用场景：应用运行和安全保障 个人信息内容：运行中进程信息（进程名和PID）和其他设备基本信息（手机型号、系统版本、系统 API 等级等） 使用目的：崩溃分析、日志收集 收集方式：系统自动收集

使用场景：推送消息给用户时 个人信息内容：设备标识符（如 Android ID、OAID、IMEI 等）和通过相关标识符生成不关联用户的推送识别 ID；设备信息（型号、系统、版本类型、网络状态、应用信息、系统语言） 使用目的：发送即时通信消息通知 收集方式：系统自动收集

使用场景：投诉 个人信息内容：账号信息、投诉信息； 使用目的：与您联系、协助您解决问题或记录相关问题的处理方案及结果 收集方式：用户主动提供

使用场景：在交互界面中查阅并选择同意/拒绝本政策时 个人信息内容：设备信息（可变 AndroidID）和日志信息（时间戳以及对应的系统、应用及协议版本信息） 使用目的：确认您可有效适用的本政策版本以保障您的合法权益 收集方式：系统应用自动采集

更多关于本应用获取和使用设备信息的说明请查看 《微企聊隐私保护指引》。
